## 🔴 우선순위 A (비교적 간단, 빠른 개선)

### A-1. 드론 영상 화면 하단 스코어보드 제거
- 드론 영상 + 야구카드 슬라이드 화면 하단의 스코어보드 및 스탯 텍스트 완전 제거
- 실제 매치업과 엇박자 나는 문제 해소
- 해당 화면은 드론 영상 + 카드슬라이드만 깔끔하게

### A-2. 문제 화면 UX 개선 (공격 타석)
- 스코어보드(주자+점수) 상단 유지
- 셋포지션 버튼 누르면 → 카테고리 스탯 바 스르륵 다시 한 번 노출
- 힌트 칸(정답 박스) 크기 대폭 확대 + 위치 상단으로 이동 (가독성 개선)
- 공격 타석에서만 적용
